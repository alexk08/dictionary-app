# Dictionary Application

Single page application for searching the meaning of English words. Also you can save the words in the section Starred Words and filter these. All session data are saved on the client side in local storage.

## Deploy: 
### [Dictionary Application](https://dictionary-app-alexk08.netlify.app/)

## Stack
- React & Redux
- Create React App
- Typescript
- SASS
- Bootstrap
- React Router
- React Sortable HOC
- [WordsAPI](https://www.wordsapi.com/)

## Usage
### `npm i`
### `npm start`
### `npm build`
